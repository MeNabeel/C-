🔵 Advanced Conditional Practice
A collection of C++ programming exercises focused on mastering conditional statements (if-else, nested conditions). These problems are designed to build strong logical thinking before moving to loops.

📋 Practice Problems
1️⃣ Number Type Checker
Take an integer input and print its type:

"Positive Even" - Number > 0 and divisible by 2

"Positive Odd" - Number > 0 and not divisible by 2

"Negative Even" - Number < 0 and divisible by 2

"Negative Odd" - Number < 0 and not divisible by 2

"Zero" - Number equals 0

Example:

text
Input: -4
Output: Negative Even
2️⃣ Valid Triangle Checker
Input 3 angles of a triangle and determine:

First check if angles sum to 180° → "Valid Triangle" or "Invalid Triangle"

If valid, also classify the triangle type:

Acute - All angles < 90°

Right - One angle = 90°

Obtuse - One angle > 90°

Example:

text
Input: 60 60 60
Output: Valid Triangle, Acute
3️⃣ Login System
Create a simple authentication system with:

Correct credentials:

Username: admin

Password: 12345

Program behavior:

If username incorrect → "Invalid Username"

If username correct but password wrong → "Wrong Password"

If both correct → "Login Successful"

Example:

text
Enter Username: admin
Enter Password: 12345
Output: Login Successful
4️⃣ Largest Between Four Numbers
Take 4 numbers as input and print the largest number using only conditional statements (no arrays, no loops).

Example:

text
Input: 10 25 5 18
Output: 25
5️⃣ Character Identifier
Input a single character and identify its type:

"Uppercase" - A to Z

"Lowercase" - a to z

"Digit" - 0 to 9

"Special Character" - Any other character

Examples:

text
Input: A
Output: Uppercase

Input: g
Output: Lowercase

Input: 5
Output: Digit

Input: #
Output: Special Character
